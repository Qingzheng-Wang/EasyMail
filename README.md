# EasyMail

<center>

<img src="https://github.com/Qingzheng-Wang/EasyMail/blob/main/image/sign_in_window.png" width="300" />
</center>

EasyMail是一个使用Python编写的电子邮件收发客户端。此项目是武汉大学计算机网络课程的大作业。

项目主要使用SMTP和POP3协议收发邮件，使用PyQt5编写用户界面，使用MySQL管理数据库。
